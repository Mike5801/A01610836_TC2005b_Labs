Laboratorio 4: Problema 6

Problema 6:

New Year is coming and you are excited to know how many minutes remain before the New Year. You know that currently the clock shows h hours and m minutes, where 0≤hh<24 and 0≤mm<60. We use 24-hour time format
Your task is to find the number of minutes before the New Year.
You know that New Year comes when the clock shows 0 hours and 0 minutes.

Input:
contains the time as two integers h and m (0≤h<24, 0≤m<60). It is guaranteed that this time is not a midnight, i.e. the following two conditions can't be met at the same time: h=0 and m=0. It is guaranteed that both h and m are given without leading zeros.

Output:
Print the answer on it — the number of minutes before the New Year.

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias:

  • Java es un lenguaje de programacióno rientado a objetos, mientras que JavaScript está basado en prototipos
  • JavaScript es débilmente tipado, significando que una misma variable puede contener diferentes tipos de información a lo largo del tiempo, mientras que Java es fuertemente tipado, lo que indica que todas las variables tienen un tipo determinado y no se pueden cambiar una vez definidas.
  • JavaScript es un lenguaje interpretado y Java es un lenguaje compilado

(Cerón, M., s.f.)

Semejanzas:

Por lo general, no existe una gran similitud entre ambos lenguajes, ya que no se relacionan uno con otro, a excepción del nombre y que parte de la sintaxis es similar. (ISDI coders, 2017)
